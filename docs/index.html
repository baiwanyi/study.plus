核心功能

打造全方位学习管理体验

从作业提交到积分兑换，AI 赋能每一个学习环节

作业提交与 AI 评分

支持 Markdown 编辑器与思维导图，DeepSeek 自动评分（A+~E），返回评语与改进建议。

积分奖惩与兑换

学习分享卡片

一键生成积分、作业、周报的分享卡片，支持多种背景，记录成长瞬间。

学迹电台 & RSS 阅读

本地开发

三步启动项目

在本地运行学迹Plus，只需简单几步

1.

安装依赖并配置环境变量

pnpm install
cp .env.example .env # 编辑 .env 填入 DeepSeek API Key
2.

初始化数据库并启动开发服务器

pnpm db:migrate
pnpm dev # 同时启动前端(5173)和后端(3001)
3.

打开浏览器访问

http://localhost:5173

更多脚本请查看 package.json，生产构建使用 pnpm build && pnpm start

需要 Node.js ≥ 18，推荐使用 pnpm 管理依赖

在 GitHub 上查看完整文档
