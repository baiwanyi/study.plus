本地开发

三步启动项目

在本地运行学迹Plus，只需简单几步

1.

安装依赖并配置环境变量

pnpm install
cp .env.example .env# 编辑 .env 填入DeepSeek API Key
2.

初始化数据库并启动开发服务器

pnpm db:migrate
pnpm dev# 同时启动前端(5173)和后端(3001)
3.

打开浏览器访问

http://localhost:5173

更多脚本请查看package.json，生产构建使用pnpm build && pnpm start

需要 Node.js ≥ 18，推荐使用 pnpm 管理依赖

在 GitHub 上查看完整文档提交 Issue
