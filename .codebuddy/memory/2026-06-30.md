# 2026-06-30

## 完成项目前后端分离重构

完成了整个项目的前后端分离架构重构，采用 pnpm workspace monorepo 架构拆分为 3 个包：

### 新目录结构
- `shared/` — 共享类型、常量、Zod Schema、纯工具函数
- `server/` — 后端 (Express + SQLite)
- `apps/` — 前端 (React + Vite)，保留原名

### 关键变更
- `shared/src/types.ts` ← `apps/lib/types.ts`
- `shared/src/constants.ts` ← `apps/lib/default.ts`
- `shared/src/weekly.ts` ← `apps/lib/weekly.ts`
- `shared/src/utils.ts` ← `apps/lib/utils.ts` 纯函数部分（formatDate, toTaskType, taskStatus, taskTypeValues 等）
- `apps/src/main.tsx` ← `apps/index.tsx`
- `apps/src/api/index.ts` ← `apps/lib/api.ts`
- `apps/src/utils/index.ts` ← `apps/lib/utils.ts` 浏览器专用部分（isAdmin, loadConfig, taskTypeColors 等），同时 re-export 所有 shared/utils 函数
- `pages/` → `apps/src/pages/`
- `apps/components/` → `apps/src/components/`
- `apps/styles/` → `apps/src/styles/`
- `public/` → `apps/public/`
- `server/src/index.ts` ← `apps/server.ts`
- `server/src/routes/` ← `apps/routes/`
- `server/src/services/` ← `apps/services/`
- `server/src/db/` ← `apps/db/`
- Root `index.html` → `apps/index.html`
- Root `vite.config.ts` → `apps/vite.config.ts`
- Root `postcss.config.js` → `apps/postcss.config.js`
- Root `eslint.config.js` → `apps/eslint.config.js`
- 根 `package.json` 简化为 workspace 编排脚本

### 注意事项
- `apps/src/utils/index.ts` re-export 了 `@shared/utils` 的所有内容，前端页面统一从 `@/utils` 导入
- 所有 import 路径已完成批量替换：`@apps/*` → `@/*` 或 `@shared/*`
- 测试目录 `__tests__/` 已创建（空目录），后续测试迁移使用
- 开发命令改为：`pnpm dev`（concurrently 启动前后端）
- 构建命令改为：`pnpm build`
- 数据库迁移命令改为：`pnpm db:migrate`
